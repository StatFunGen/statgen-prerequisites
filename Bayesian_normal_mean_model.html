Bayesian Normal Mean Model#

The Bayesian normal mean model combines our prior belief about a normal distribution’s mean with observed data, resulting in a posterior mean that’s a precision-weighted average of the prior mean and sample mean.

Graphical Summary#

Key Formula#

Under the Bayesian regression setting, we have $\( \mathbf{Y} = \mathbf{X} \beta + \boldsymbol{\varepsilon}, \quad \boldsymbol{\varepsilon} \sim \mathcal{N}(0, \sigma^2 \mathbf{I}) \)\( where we place a **prior distribution** on the regression coefficients: \)\beta \sim \mathcal{N}(\beta_0, \Sigma_0)$.

  • \( \mathbf{Y} \in \mathbb{R}^{N \times 1} \): phenotype vector

  • \( \mathbf{X} \in \mathbb{R}^{N \times 1} \): genotype vector (e.g., 0, 1, 2 for additive coding)

  • \( \beta \in \mathbb{R} \): a scalar, genetic effect size (regression coefficient)

  • \( \boldsymbol{\varepsilon} \in \mathbb{R}^{N \times 1} \): residual error, assumed i.i.d. Gaussian

Technical Details#

Note that we switch from \(\beta\) to \(\beta\) as we now consider \(\beta\) a random variable to estimate, but conceptually they are the same thing – effect of \(X\) on \(Y\).

Model Assumptions#

In the Bayesian normal mean model, we assume the variable \(Y\) follows uni-variate normal model \(\mathcal{N}(X\beta, \sigma^2)\)

\[ Y \sim \mathcal{N}(X\beta, \sigma^2) \]

where:

  • \( \beta \) is the unknown mean parameter.

  • \( \sigma^2 \) is the known variance.

  • \(X\) is the genotype value (fixed) for each individual

Since \(X\) is fixed, then the question becomes what can we learn about the unknown parameter \(\beta\).

Prior Distribution#

A common choice for the prior on \( \beta \) is a normal distribution:

\[ \beta \sim \mathcal{N}(\beta_0, \sigma_0^2) \]

where \( \beta_0 \) is the prior mean and \( \sigma_0^2 \) is the prior variance.

The precision is defined as \(\tau_0 = \frac{1}{\sigma_0^2}\). In other words, the prior of \(\beta\) can be rewritten as:

\[ \beta \sim \mathcal{N}(\beta_0, \frac{1}{\tau_0}) \]

Therefore, the probability density function (PDF) is

\[\begin{split} \begin{align} p(\beta)&\propto \exp\left(-\frac{1}{2}\tau_0\beta^2 + \tau_0\beta_0 \beta\right) \\ &\propto \exp\left(-\frac{1}{2}\tau_0 (\beta - \beta_0)^2\right) \end{align} \end{split}\]

(Here, the 0 subscript is used to indicate that \(\beta_0\) and \(\sigma_0\) are parameters in the prior.)

Likelihood#

The likelihood of the data given \(\beta\) and \(\sigma^2\) is:

\[ Y\mid\beta, \sigma^2 \sim \mathcal{N}(\beta, \sigma^2) \]

The probability density function (PDF) is:

\[\begin{split} \begin{align} p(Y\mid\beta) &\propto \exp\left(-\frac{1}{2} \tau (Y^2 - 2\beta Y)\right)\\ &\propto \exp\left(-\frac{1}{2} \tau (Y - \beta)^2\right) \end{align} \end{split}\]

where \(\tau = \frac{1}{\sigma^2}\) is the precision.

Posterior Distribution#

Using Bayes’ theorem, the posterior distribution of \( \beta \) is given by:

\[ p(\beta|Y) \propto p(Y|\beta) p(\beta) \]

Expanding the probability density functions:

\[\begin{split} \begin{align} p(\beta \mid Y) &\propto \exp\left(-\frac{1}{2} \tau (Y - \beta)^2\right) \exp\left(-\frac{1}{2} \tau_0 (\beta - \beta_0)^2\right) \\ &\propto \exp\left(-\frac{1}{2} \tau (Y^2 - 2Y\beta + \beta^2) - \frac{1}{2} \tau_0 (\beta^2 - 2\beta \beta_0 + \beta_0^2)\right) \\ &\propto \exp\left(-\frac{1}{2} (\tau + \tau_0) \beta^2 + (\tau Y + \tau_0 \beta_0) \beta \right) \end{align} \end{split}\]

Recognizing the quadratic form, the posterior distribution follows a normal distribution:

\[ \beta \mid Y \sim \mathcal{N}\left( \beta_1, \sigma_1^2 \right) \]

where

\[ \beta_1 = \frac{\tau Y + \tau_0 \beta_0}{\tau + \tau_0}, \quad \sigma_1^2 = \frac{1}{\tau + \tau_0} \]

Characteristics#

  • Bayes’ theorem combines these components: \(P(\beta|D) \propto P(D|\beta) P(\beta)\).

  • The prior represents our belief about the parameter before seeing any data.

  • The likelihood represents how likely we are to observe the data given the parameter.

  • The posterior is the updated belief about the parameter after seeing the data.

    • the Posterior precision (\(\tau_1\)) is the sum of the Data precision (\(\tau\)) and the Prior precision (\(\tau_0\)). This makes sense: the more precise your data, and the more precise your prior information, the more precise your posterior information. Also, this means that the data always improves your posterior precision compared with the prior: noisy data (small \(\tau\)) improves it only a little, whereas precise data improves it a lot.

    • If we rewrite the posterior mean as:

    \[ \beta_1=w\sum_i^n x_i+(1−w)\beta_0 \]

    where \(w=\tau(\tau+\tau_0)\). Thus \(\beta_1\) is a weighted average of the sum of the data \(x_i\) and the prior mean \(\beta_0\). And the weights depend on the relative precision of the data and the prior. If the data are precise compared with the prior (\(\tau>>\tau_0\)) then the weight \(w\) will be close to 1 and the posterior mean will be close to the data.

Example#

This R example demonstrates a Bayesian normal mean model for updating beliefs about a population parameter using observed data. The analysis starts with a prior belief that the true mean (\(\beta\)) follows a normal distribution with mean 0.1 and variance 1. After observing height measurements from five individuals, the code applies Bayesian updating rules to compute the posterior distribution, which combines the prior information with the likelihood of the observed data. The example illustrates how the posterior mean shifts toward the sample average while the posterior variance decreases (indicating increased certainty), and visualizes both the prior and posterior distributions along with the observed data points to show how Bayesian inference updates our beliefs in light of new evidence.

  • Requirement:

    • Bayes rule

    • OLS

# Clear the environment
rm(list = ls())

Generate Observed Data#

First we define the true parameters, which are unknown in practice, but here we are simulating the data. Then we generate the observed data.

# True genetic effect of one variant on height
beta_true <- 4.5      # cm per allele (this is what we want to estimate)
sigma_epsilon <- 8    # residual standard deviation (cm)

n_individuals <- 10           # number of individuals
height_baseline <- 170       # population mean height (cm)

# Generate genotypes (0, 1, or 2 copies of the variant)
# Assuming MAF = 0.3 (Minor Allele Frequency)
maf <- 0.3
p_genotypes <- c((1-maf)^2, 2*maf*(1-maf), maf^2)  # P(0), P(1), P(2)

genotypes <- sample(0:2, size = n_individuals, replace = TRUE, prob = p_genotypes)

# Calculate genetic contributions and add environmental noise
genetic_effects <- genotypes * beta_true
environmental_effects <- rnorm(n_individuals, mean = 0, sd = sigma_epsilon)
observed_heights <- height_baseline + genetic_effects + environmental_effects

# Create dataset
data <- data.frame(
  individual = 1:n_individuals,
  genotype = genotypes,
  height = observed_heights,
  genetic_effect = genetic_effects,
  environmental_effect = environmental_effects
)

Bayesian Inference#

Prior Distribution#

We assume a normal prior for \(\beta\):

\[ \beta \sim \mathcal{N}(\beta_0, \sigma_0^2) \]

where:

  • \(\beta_0 = 0\) (prior mean),

  • \(\sigma_0^2 = 25\) (prior variance), leading to a precision \(\tau_0 = 1/\sigma_0^2\).

# Prior belief about genetic effect (weakly informative)
beta_prior_mean <- 0         # center at zero (no prior knowledge)
beta_prior_var <- 25         # prior variance (SD = 5 cm per allele)
beta_prior_precision <- 1 / beta_prior_var

Likelihood#

cat("Likelihood for each individual i:\n")
cat("height_i | beta ~ N(", height_baseline, " + beta * genotype_i,", sigma_epsilon^2, ")\n")
cat("Known residual variance:", sigma_epsilon^2, "cm^2\n")
cat("Residual precision:", 1/sigma_epsilon^2, "\n\n")

cat("The likelihood is a function of beta (unknown parameter):\n")
for(i in 1:min(3, n_individuals)) {
  cat("Individual", i, ": height =", round(data$height[i], 1), 
      "cm, genotype =", data$genotype[i], "\n")
  cat("  Expected height = ", height_baseline, " + beta *", data$genotype[i], 
      "=", height_baseline, "+ beta *", data$genotype[i], "\n")
  cat("  Likelihood contribution is proportional to exp(-0.5 * (", round(data$height[i], 1), 
      "- (", height_baseline, "+ beta *", data$genotype[i], "))^2 /", sigma_epsilon^2, ")\n")
}
cat("...\n")
cat("The likelihood function tells us how plausible different values of beta are,\n")
cat("given the observed data. We don't know the true beta - that's what we're estimating!\n\n")

A sufficient statistic T(data) contains all the information about a parameter that is available in the dataset. Remarkably, for our normal model, ALL information about β from n individuals can be compressed into just two numbers:

  • \(T_1\) = Σ(genotype_i²)

  • \(T_2\) = Σ(genotype_i · (height_i - baseline))

Due to the exponential family property of the normal distribution, the likelihood can be rewritten as:

\[ L(\beta|\text{data}) = L(\beta|T_1, T_2, N) \]

This means we can throw away all the raw data and work only with \(T_1\), \(T_2\), and \(N\), without losing any information about \(\beta\)!

Sufficient statistics are particularly powerful when combined with conjugate priors. For the normal-normal model:

  • Prior: \(\beta \sim N(\beta_\text{prior}, \sigma_\text{prior}^2)\)

  • Likelihood: data | \(\beta\) ~ Normal with sufficient statistics \(T_1, T_2\)

  • Posterior: \(\beta | \text{data} \sim N(\beta_\text{posterior}, \sigma_\text{posterior}^2))\) with simple update formulas

This creates a beautiful mathematical framework where:

  1. We can summarize any amount of data with just a few numbers

  2. Bayesian updating becomes elementary algebra

  3. The posterior has the same functional form as the prior

Sufficient statistics transform Bayesian inference from a computationally intensive, data-heavy process into an elegant, efficient calculation. They are the reason why certain Bayesian models can scale to massive datasets while maintaining mathematical rigor and computational tractability.

Without sufficient statistics, modern Bayesian genomics, machine learning, and large-scale data analysis would be practically impossible.

# For conjugate normal-normal model, we need:
# Sum of (genotype_i^2) and Sum of (genotype_i * (height_i - baseline))

sum_genotype_squared <- sum(genotypes^2)
sum_genotype_height <- sum(genotypes * (observed_heights - height_baseline))

Posterior#

Using Bayesian updating, the posterior distribution is given by:

\[ \beta | Y \sim \mathcal{N}(\beta_1, \sigma_1^2) \]

where:

  • Posterior precision: \(\tau_1 = \tau_0 + n \tau\)

  • Posterior mean: \(\beta_1 = \frac{\tau_0 \beta_0 + \tau \sum Y}{\tau_1}\)

  • Posterior standard deviation: \(\sigma_1 = \sqrt{\frac{1}{\tau_1}}\)

# Posterior precision = prior precision + likelihood precision
likelihood_precision <- sum_genotype_squared / sigma_epsilon^2
posterior_precision <- beta_prior_precision + likelihood_precision

# Posterior variance
posterior_variance <- 1 / posterior_precision

# Posterior mean (precision-weighted average)
posterior_mean <- (beta_prior_precision * beta_prior_mean + 
                  likelihood_precision * (sum_genotype_height / sum_genotype_squared)) / 
                  posterior_precision

cat("Bayesian updating formulas:\n")
cat("Posterior precision = prior precision + likelihood precision\n")
cat("                   =", round(beta_prior_precision, 4), "+", round(likelihood_precision, 4), 
    "=", round(posterior_precision, 4), "\n\n")

cat("Posterior variance = 1 / posterior precision =", round(posterior_variance, 4), "\n\n")

cat("Posterior mean = weighted average of prior and data\n")
cat("Weight on prior:", round(beta_prior_precision / posterior_precision, 3), "\n")
cat("Weight on data:", round(likelihood_precision / posterior_precision, 3), "\n\n")

cat("POSTERIOR RESULTS:\n")
cat("Posterior distribution: beta | data ~ N(", round(posterior_mean, 3), ",", 
    round(posterior_variance, 4), ")\n")
cat("Posterior mean:", round(posterior_mean, 3), "cm per allele\n")
cat("Posterior standard deviation:", round(sqrt(posterior_variance), 3), "cm per allele\n")
cat("True value (for comparison):", beta_true, "cm per allele\n\n")

(optional) Compare with MLE#

# Maximum likelihood estimator (same as OLS for normal model)
# beta_hat = (X'X)^(-1) X'Y where X is design matrix and Y is outcome vector

# For our simple model: height_i = baseline + beta * genotype_i + epsilon_i
# The MLE is: beta_hat = sum(genotype_i * (height_i - baseline)) / sum(genotype_i^2)

beta_mle <- sum_genotype_height / sum_genotype_squared
se_mle <- sqrt(sigma_epsilon^2 / sum_genotype_squared)

cat("Maximum Likelihood Estimator (MLE):\n")
cat("beta_hat =", round(beta_mle, 4), "cm per allele\n")
cat("Standard error:", round(se_mle, 4), "cm per allele\n")

# 95% confidence interval
ci_lower <- beta_mle - 1.96 * se_mle
ci_upper <- beta_mle + 1.96 * se_mle

cat("95% Confidence Interval: [", round(ci_lower, 2), ",", round(ci_upper, 2), "] cm per allele\n")

Standard error: 3.266 cm per allele

Graphical Summary#

library(ggplot2)
library(dplyr)

# Set parameters
# Prior: N(mu_0, sigma_0^2)
mu_0 <- 0      # Prior mean
sigma_0 <- 1   # Prior standard deviation (smaller = more informative prior)

# Data (observed values)
y_obs <- c(1.2, 2.1, 1.8, 2.3, 1.9)  # Sample data
n <- length(y_obs)
y_bar <- mean(y_obs)  # Sample mean
sigma <- 4  # Known likelihood standard deviation (larger = less informative data)

# Calculate posterior parameters (conjugate normal-normal case)
# Posterior precision = prior precision + likelihood precision
tau_0 <- 1 / sigma_0^2    # Prior precision
tau_likelihood <- n / sigma^2  # Likelihood precision
tau_1 <- tau_0 + tau_likelihood  # Posterior precision

# Posterior parameters
sigma_1 <- sqrt(1 / tau_1)  # Posterior standard deviation
mu_1 <- (tau_0 * mu_0 + tau_likelihood * y_bar) / tau_1  # Posterior mean

# Create data for plotting
beta_range <- seq(-4, 6, length.out = 1000)

# Calculate densities
prior_density <- dnorm(beta_range, mu_0, sigma_0)
likelihood_density <- dnorm(beta_range, y_bar, sigma/sqrt(n))  # Likelihood as function of beta
posterior_density <- dnorm(beta_range, mu_1, sigma_1)

# Combine into data frame
plot_data <- data.frame(
  beta = rep(beta_range, 3),
  density = c(prior_density, likelihood_density, posterior_density),
  distribution = rep(c("Prior", "Likelihood", "Posterior"), each = length(beta_range))
)

# Create the plot
p <- ggplot(plot_data, aes(x = beta, y = density, color = distribution, linetype = distribution)) +
  geom_line(size = 1.2) +
  scale_color_manual(values = c("Prior" = "tomato", 
                               "Likelihood" = "#000080", 
                               "Posterior" = "darkgreen")) +
  scale_linetype_manual(values = c("Prior" = "dashed", 
                                  "Likelihood" = "dotted", 
                                  "Posterior" = "solid")) +
  labs(
    y = "Density",
    x = expression(beta),
    color = "Distribution",
    linetype = "Distribution"
  ) +
  theme_minimal(base_size = 20) +
  theme(
    plot.title = element_blank(),
    axis.title.y = element_text(face = "bold"),
    axis.title.x = element_text(face = "bold"),
    axis.text.x = element_text(face = "bold"),
    axis.text.y = element_text(face = "bold"),
    legend.title = element_blank(),
    legend.text = element_text(face = "bold"),
    legend.position = "bottom"
  ) +
  guides(color = guide_legend(override.aes = list(size = 1.5)))

# Add vertical lines for means
p <- p + 
  geom_vline(xintercept = mu_0, color = "tomato", linetype = "dashed", alpha = 0.7) +
  geom_vline(xintercept = y_bar, color = "skyblue", linetype = "dotted", alpha = 0.7) +
  geom_vline(xintercept = mu_1, color = "darkgreen", linetype = "solid", alpha = 0.7)

# Display plot
print(p)

# Save plot to file
ggsave("./cartoons/Bayesian_normal_mean_model.png", plot = p,
       width = 10, height = 6, 
       bg = "transparent",
       dpi = 300)
