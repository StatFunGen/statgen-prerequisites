Bayesian Normal Mean Model#

The Bayesian normal mean model takes your initial guess about a parameter and adjusts it based on new data, with the final estimate being a weighted average that depends on how confident you were initially versus how informative the data is.

Graphical Summary#

Key Formula#

Under the Bayesian regression setting, we have

\[ \mathbf{Y} = \mathbf{X} \beta + \boldsymbol{\varepsilon}, \quad \boldsymbol{\varepsilon} \sim \mathcal{N}(0, \sigma^2 \mathbf{I}) \]

where we place a prior distribution on the regression coefficient: \(\beta \sim \mathcal{N}(\beta_0, \sigma_0^2)\).

  • \( \mathbf{Y} \in \mathbb{R}^{N \times 1} \): phenotype vector

  • \( \mathbf{X} \in \mathbb{R}^{N \times 1} \): genotype vector (e.g., 0, 1, 2 for additive coding)

  • \( \beta \in \mathbb{R} \): a scalar, genetic effect size (regression coefficient)

  • \( \boldsymbol{\varepsilon} \in \mathbb{R}^{N \times 1} \): residual error, assumed i.i.d. Gaussian

Technical Details#

Model Assumptions#

In the Bayesian normal mean model, each observation \(Y_i\) follows a univariate normal distribution:

\[ Y_i \sim \mathcal{N}(X_i\beta, \sigma^2) \quad \text{for } i = 1, 2, \ldots, N \]

where:

  • \( \beta \) is the unknown mean parameter (genetic effect)

  • \( \sigma^2 \) is the known variance

  • \(X_i\) is the genotype value (fixed) for individual \(i\)

Since \(X_i\) values are fixed, the question becomes: what can we learn about the unknown parameter \(\beta\)?

Prior Distribution#

A common choice for the prior on \( \beta \) is a normal distribution:

\[ \beta \sim \mathcal{N}(\beta_0, \sigma_0^2) \]

where \( \beta_0 \) is the prior mean and \( \sigma_0^2 \) is the prior variance.

The precision is defined as \(\tau_0 = \frac{1}{\sigma_0^2}\). In other words, the prior of \(\beta\) can be rewritten as:

\[ \beta \sim \mathcal{N}\left(\beta_0, \frac{1}{\tau_0}\right) \]

Therefore, the probability density function (PDF) is:

\[\begin{split} \begin{align} p(\beta) &\propto \exp\left(-\frac{1}{2}\tau_0(\beta - \beta_0)^2\right) \\ &\propto \exp\left(-\frac{1}{2}\tau_0\beta^2 + \tau_0\beta_0 \beta\right) \end{align} \end{split}\]

Likelihood#

The likelihood of the data given \(\beta\) and \(\sigma^2\) is:

\[ Y_i\mid\beta, \sigma^2 \sim \mathcal{N}(X_i\beta, \sigma^2) \quad \text{for } i = 1, 2, \ldots, N \]

For all observations, the likelihood is:

\[\begin{split} \begin{align} p(\mathbf{Y}\mid\beta) &\propto \exp\left(-\frac{1}{2\sigma^2} \sum_{i=1}^N (Y_i - X_i\beta)^2\right)\\ &\propto \exp\left(-\frac{1}{2} \tau \sum_{i=1}^N (Y_i - X_i\beta)^2\right) \end{align} \end{split}\]

where \(\tau = \frac{1}{\sigma^2}\) is the precision.

Posterior Distribution#

Using Bayes’ theorem, the posterior distribution of \( \beta \) is given by:

\[ p(\beta|\mathbf{Y}) \propto p(\mathbf{Y}|\beta) p(\beta) \]

The posterior distribution follows a normal distribution:

\[ \beta \mid \mathbf{Y} \sim \mathcal{N}\left( \beta_1, \sigma_1^2 \right) \]

where the parameters are:

\[ \beta_1 = \frac{\tau \sum_{i=1}^N X_i Y_i + \tau_0 \beta_0}{\tau \sum_{i=1}^N X_i^2 + \tau_0}, \quad \sigma_1^2 = \frac{1}{\tau \sum_{i=1}^N X_i^2 + \tau_0} \]

These can also be written in terms of sufficient statistics:

\[ \beta_1 = \frac{\tau T_2 + \tau_0 \beta_0}{\tau T_1 + \tau_0}, \quad \sigma_1^2 = \frac{1}{\tau T_1 + \tau_0} \]

where \(T_1 = \sum_{i=1}^N X_i^2\) and \(T_2 = \sum_{i=1}^N X_i Y_i\) are the sufficient statistics.

Interpretation as Weighted Average#

The posterior mean can be rewritten as a weighted average:

\[\beta_1 = w \cdot \frac{T_2}{T_1} + (1-w) \cdot \beta_0\]

where:

\[w = \frac{\tau T_1}{\tau T_1 + \tau_0}\]

This shows \(\beta_1\) balances the data’s estimate \(\frac{T_2}{T_1}\) (MLE) and the prior mean \(\beta_0\), with weights determined by relative precision:

  • Precise data (\(\tau T_1 >> \tau_0\)): \(w \approx 1\), posterior follows data

  • Precise prior (\(\tau_0 >> \tau T_1\)): \(w \approx 0\), posterior follows prior

  • Equal precision (\(\tau T_1 = \tau_0\)): \(w = 0.5\), equal weighting

Example#

In our previous likelihood analysis, we found that among our three theories about genetic effects (\(\beta=0\), \(\beta=0.5\), \(\beta=1.0\)), the data supported \(\beta=0.5\) most strongly.

But what if we don’t want to compare specific models, but instead have some prior knowledge about the parameter \(\beta\) before seeing the data?

For example, maybe from previous studies we generally believe that genetic effects are usually small, centering around zero with some uncertainty. This is fundamentally different from testing discrete hypotheses - instead of asking “which of these three values is most likely?”, we’re asking “given what we knew before, what do we believe now?”

This example shows how to implement a Bayesian normal mean model, where we treat \(\beta\) as a continuous parameter with a prior distribution. Instead of probabilities for just three specific values, we get a complete posterior distribution showing how plausible every possible effect size is after combining our prior knowledge with the data.

Using the same genetic data (true effect \(\beta=0.4\)), we’ll see how prior beliefs get updated to give us a posterior distribution. This answers richer questions: What’s our best guess for the effect size? How uncertain are we? What’s the probability the effect is positive or larger than some threshold?

The beautiful thing about the normal-normal conjugate model is that everything works out analytically - clean formulas show exactly how prior beliefs and data evidence combine.

Generate Simulated Data#

First, let’s recreate the exact same genetic data from our previous likelihood analysis.

# Clear the environment
rm(list = ls())
set.seed(19)  # For reproducibility

# Generate genotype data for 5 individuals at 1 variant
N <- 5
genotypes <- c("CC", "CT", "TT", "CT", "CC")  # Individual genotypes
names(genotypes) <- paste("Individual", 1:N)

# Define alternative allele
alt_allele <- "T"

# Convert to additive genotype coding (count of alternative alleles)
Xraw_additive <- numeric(N)
for (i in 1:N) {
  alleles <- strsplit(genotypes[i], "")[[1]]
  Xraw_additive[i] <- sum(alleles == alt_allele)
}
names(Xraw_additive) <- names(genotypes)

# Standardize genotypes
X <- scale(Xraw_additive, center = TRUE, scale = TRUE)[,1]

# Set true beta and generate phenotype data
true_beta <- 0.4
true_sd <- 1.0

# Generate phenotype with true effect
Y <- X * true_beta + rnorm(N, 0, true_sd)

Bayesian Normal Mean Model Setup#

Now we specify our Bayesian model. The key insight is that instead of testing specific values of \(\beta\), we place a continuous prior distribution over all possible values.

Our model has three components:

  1. Likelihood: How likely is our data given a specific \(\beta\)? $\(Y_i | \beta \sim \mathcal{N}(X_i \beta, \sigma^2)\)$

  2. Prior: What did we believe about \(\beta\) before seeing data? $\(\beta \sim \mathcal{N}(\beta_0, \sigma_0^2)\)$

  3. Posterior: What do we believe about \(\beta\) after seeing data? $\(\beta | \mathbf{Y} \sim \mathcal{N}(\beta_1, \sigma_1^2)\)$

# Prior specification for beta
beta_0 <- 0        # Prior mean (no effect expected a priori)
sigma_0 <- 0.6     # Prior standard deviation (moderate uncertainty)
tau_0 <- 1/sigma_0^2  # Prior precision

# Known error variance (in practice, this might also be estimated)
sigma2 <- true_sd^2
tau <- 1/sigma2    # Likelihood precision

print(paste0("Prior: beta ~ N(", beta_0, ", ", sigma_0^2, ")"))
print(paste0("Prior precision tau_0: ", round(tau_0, 3)))

[1] "Prior: beta ~ N(0, 0.36)"
[1] "Prior precision tau_0: 2.778"

Calculate the Posterior Distribution#

The magic of conjugate priors is that when we combine a normal prior with a normal likelihood, we get a normal posterior with closed-form solutions.

Sufficient Statistics: All We Need#

A remarkable property of the normal distribution is that all information about β in our dataset can be compressed into just two numbers:

  • \(T_1 = \sum_{i=1}^N X_i^2\) = sum of squared genotypes

  • \(T_2 = \sum_{i=1}^N X_i Y_i\) = sum of genotype × phenotype products

In matrix notation: \(T_1 = \mathbf{X}^T\mathbf{X}\) and \(T_2 = \mathbf{X}^T\mathbf{Y}\)

These sufficient statistics contain everything the data tells us about β - we could delete all the raw observations and still get the exact same posterior!

Posterior Formulas#

The posterior parameters are: $\(\beta_1 = \frac{\tau T_2 + \tau_0 \beta_0}{\tau T_1 + \tau_0} = \frac{\tau \sum_{i=1}^N X_i Y_i + \tau_0 \beta_0}{\tau \sum_{i=1}^N X_i^2 + \tau_0}\)$

\[\sigma_1^2 = \frac{1}{\tau T_1 + \tau_0} = \frac{1}{\tau \sum_{i=1}^N X_i^2 + \tau_0}\]

This shows how the posterior mean \(\beta_1\) is a precision-weighted average of the prior mean \(\beta_0\) and the data’s estimate \(T_2/T_1\).

# Calculate sufficient statistics from the data
sum_X_squared <- sum(X^2)    # T1: captures variability in genotypes
sum_XY <- sum(X * Y)         # T2: captures genotype-phenotype relationship

cat("Sufficient statistics (all info about beta):\n")
cat("T1 = sum(X^2) =", round(sum_X_squared, 3), "\n")
cat("T2 = sum(XY) =", round(sum_XY, 3), "\n")
cat("Data's estimate of beta = T2/T1 =", round(sum_XY/sum_X_squared, 3), "\n\n")

Sufficient statistics (all info about beta):
T1 = sum(X^2) = 4 

T2 = sum(XY) = 1.268 
Data's estimate of beta = T2/T1 = 0.317

Then we can calculate the posterior:

# Calculate posterior parameters
beta_1 <- (tau * sum_XY + tau_0 * beta_0) / (tau * sum_X_squared + tau_0)
sigma_1_squared <- 1 / (tau * sum_X_squared + tau_0)
sigma_1 <- sqrt(sigma_1_squared)

print(paste0("Posterior: beta | Y ~ N(", round(beta_1, 3), ", ", round(sigma_1_squared, 4), ")"))

[1] "Posterior: beta | Y ~ N(0.187, 0.1475)"

We take a look at how the prior and data are weighted:

# Show how prior and data are weighted
prior_weight <- tau_0 / (tau_0 + tau * sum_X_squared)
data_weight <- (tau * sum_X_squared) / (tau_0 + tau * sum_X_squared)
cat("Prior weight:", round(prior_weight, 3), "\n")
cat("Data weight:", round(data_weight, 3), "\n")

Prior weight: 0.41 
Data weight: 0.59

Graphical Summary#

library(ggplot2)
library(dplyr)

# Set parameters
# Prior: N(mu_0, sigma_0^2)
mu_0 <- 0      # Prior mean
sigma_0 <- 1   # Prior standard deviation (smaller = more informative prior)

# Data (observed values)
y_obs <- c(1.2, 2.1, 1.8, 2.3, 1.9)  # Sample data
n <- length(y_obs)
y_bar <- mean(y_obs)  # Sample mean
sigma <- 4  # Known likelihood standard deviation (larger = less informative data)

# Calculate posterior parameters (conjugate normal-normal case)
# Posterior precision = prior precision + likelihood precision
tau_0 <- 1 / sigma_0^2    # Prior precision
tau_likelihood <- n / sigma^2  # Likelihood precision
tau_1 <- tau_0 + tau_likelihood  # Posterior precision

# Posterior parameters
sigma_1 <- sqrt(1 / tau_1)  # Posterior standard deviation
mu_1 <- (tau_0 * mu_0 + tau_likelihood * y_bar) / tau_1  # Posterior mean

# Create data for plotting
beta_range <- seq(-4, 6, length.out = 1000)

# Calculate densities
prior_density <- dnorm(beta_range, mu_0, sigma_0)
likelihood_density <- dnorm(beta_range, y_bar, sigma/sqrt(n))  # Likelihood as function of beta
posterior_density <- dnorm(beta_range, mu_1, sigma_1)

# Combine into data frame
plot_data <- data.frame(
  beta = rep(beta_range, 3),
  density = c(prior_density, likelihood_density, posterior_density),
  distribution = rep(c("Prior", "Likelihood", "Posterior"), each = length(beta_range))
)

# Create the plot
p <- ggplot(plot_data, aes(x = beta, y = density, color = distribution, linetype = distribution)) +
  geom_line(size = 1.2) +
  scale_color_manual(values = c("Prior" = "tomato", 
                               "Likelihood" = "#000080", 
                               "Posterior" = "darkgreen")) +
  scale_linetype_manual(values = c("Prior" = "dashed", 
                                  "Likelihood" = "dotted", 
                                  "Posterior" = "solid")) +
  labs(
    y = "Density",
    x = expression(beta),
    color = "Distribution",
    linetype = "Distribution"
  ) +
  theme_minimal(base_size = 20) +
  theme(
    plot.title = element_blank(),
    axis.title.y = element_text(face = "bold"),
    axis.title.x = element_text(face = "bold"),
    axis.text.x = element_text(face = "bold"),
    axis.text.y = element_text(face = "bold"),
    legend.title = element_blank(),
    legend.text = element_text(face = "bold"),
    legend.position = "bottom"
  ) +
  guides(color = guide_legend(override.aes = list(size = 1.5)))

# Add vertical lines for means
p <- p + 
  geom_vline(xintercept = mu_0, color = "tomato", linetype = "dashed", alpha = 0.7) +
  geom_vline(xintercept = y_bar, color = "skyblue", linetype = "dotted", alpha = 0.7) +
  geom_vline(xintercept = mu_1, color = "darkgreen", linetype = "solid", alpha = 0.7)

# Display plot
print(p)

# Save plot to file
ggsave("./cartoons/Bayesian_normal_mean_model.png", plot = p,
       width = 10, height = 6, 
       bg = "transparent",
       dpi = 300)
